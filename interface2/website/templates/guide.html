Guide d'utilisation du site


Guide de la page de données

Cette page n'est pour l'instant qu'une page d'affichage des données voici les explications qui permettent de l'utiliser correctement.
Premièrement, il est impératif de disposer d'un fichier csv pour pouvoir le visualiser, pour se faire, ouvrez le logiciel de traitement puis allez dans vos fichiers locaux en suivant le chemin suivant D:\METABO\Data
Allez ensuite dans le dossier correspondant au patient recherché puis sélectionnez un fichier xms.
Une fois ce fichier ouvert cliquez en haut gauche sur File --> Export --> Active chromatogram --> ASCII File
Vous venez de créer un fichier csv qui sera utilisable sur ce site/logiciel.
Retournez à présent sur le page de données, sélectionnez le bouton "Choisir le fichier" puis transférer le fichier csv que vous venez de créer.
Appuyez finalement sur le bouton "Process" vous pouvez faire que le tableau s'affiche proprement au dessous de la page.
Si vous désirez visualisez un nouveau fichier il faudra néanmoins rechargez la page avant de répétez les actions précédentes.

Guide de la page d'analyse

Ici vous allez pouvoir analyser le résultat que l'intelligence articielle a établi sur votre fichier patient.
L'utililisation de cette page est relativement similaire à celle de la page de données, il suffit de sélectionner le fichier csv correspondant puis de cliquez sur le bouton "Process".
Se référer au guide de la page de données si vous ne trouvez pas le fichier csv recherché.

Guide de la page d'ajout

Cette page permet l'ajout d'une donnée à un dossier particulier, le dossier d'entrainement du système de l'intelligence artificielle.
